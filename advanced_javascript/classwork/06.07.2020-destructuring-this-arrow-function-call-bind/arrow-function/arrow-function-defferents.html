Отличия стрелочных функций от обычных

  1. Всегда function expression
  2. Другой синтаксис - вместо слово function перед списком аргументов => после его
  3. Если тело функции состоит из одной строки, о можно не писать открывающую и закрывающую скобки и return
  4. Если аргумент один, его можно не заключать в круглые скобки
  5. Нельзя вызвать через new
  6. Нет псевдомассива arguments
