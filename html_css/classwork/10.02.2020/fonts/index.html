Есть 3 способа подключения шрифтов:

  1. Через тег link внутри тега head сайта (обычно так подключают шрифты в виде ссылок на онлайн-хранилища, например - Google Fonts).
  2. Через тег @import в таблице стилей
  3. Через тег @font-face в таблице стилей (подходит в основном только для скачаных шрифтов)
  4. Через тег @font-face в таблице стилей (подходит в основном только для скачаных шрифтов)
