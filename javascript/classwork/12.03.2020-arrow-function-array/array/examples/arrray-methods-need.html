Список самых востребованных методов массивов, которые нужно знать

const str = "gvyunjm ijijm inmikm" const arr = str.split(" ").join(" ") const arr=[1,2,3,4] const arr1=arr console.log(arr===arr1) //true const arr2= [...arr] console.log(arr===arr2) //false const arr3= [{name: "a"},{name: "b"},{name: "c"},{name: "v"},{name: "k"},{name: "l"},{name: "o"},{name: "p"}]
  • push()
  • pop()
  • shift()
  • unshift()
  • indexOf(a)
  • includes()
  • find()
  • findIndex((el,index,array)=>el.name==="a")
  • slice(5,6)
  • splice(5,1,)
  • forEach()
  • filter()
  • map()
  • split(" ") - это метод не массива, а строки, но он создает массив
  • join()
  • reduce()
Дополнительно:
  • Object.keys(objName) - возвращает массив ключей объекта (массив, содержищий имена всех свойств объекта)
  • Object.values(objName) - возвращает массив значенй свойств объекта (массив, содержищий все значения всех свойств объекта)
