Чтобы создать функцию, нужно:

  • Определится с именем
  • Подумать, какие у фунции будут аргументы: что вам нужно передать в функцию чтобы получить от нее нужный вам ответ.
  • Подумать о том, что она будет возвращать (какой ответ выдавать)

Как правильно называть функции:

  • Имя функции - всегда глагол
  • is: - является ли чем-то. Возрвращает true или false
  • calc: возвращает число на основе формулы
  • get: возвращает что-то, как правило не число, например, инициалы
  • convert: превращает одно значение в другое
  • show: выводим что-то на экран
