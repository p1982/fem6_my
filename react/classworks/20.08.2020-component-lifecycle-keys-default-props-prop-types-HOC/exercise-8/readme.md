### Создать HOC "список товаров"

В реальной жизни верстка одинаковых по функционалу модулей может принципиально отличаться, но методы и состояние будут теми же. А значит их можно переиспользовать. Возьмите задачу 4 и сделайте на ее основе HOC (High Order Component), вынеся в него повторяющееся состояние и и методы работы с ним.
