### Задание: форма добавления товара на Formik

Сделайте с помощью Formik форму создания товара и его 
вывод на экран. В ней должны быть
такие поля:
- название товара;
- артикул (8 цифр);
- цена;
- категория товара (выпадающий список из таких вариантов:
Ноутбуки, Планшеты, Мобильные телефоны);
- Акционный ли товар (это переключатель Да/Нет);
- доставка в регионы (Да/Нет, тип поля - галочка); 
- краткое описание товара (многострочное поле ввода, на 5 строк).

Дополнительные условия:
- все поля - обязательные;
- свойства полей вынесите в отдельный файл `fields.js`;
- после того, как пользователь нажал кнопку Submit, под формой в таблице
должен появляется товар, где каждый столбец таблицы - введенное пользователем значение.
